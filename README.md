## To start app:

Clone this repo.

Run commands:

`cd threeJSTT`

`npm i`

`npm run start` 

Enter `http://localhost:3000/` in your browser.

You should see something like that:

![Demonstration](https://github.com/MrDmitry-dev/threeJSTT/blob/master/screenOfTT.jpg)

Click the button to see it rotates.
